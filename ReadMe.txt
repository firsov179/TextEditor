================= FIRE WORD  ==========================
-------------------------------------------------------

При создании/открытии файла имя пишется без расширения.

Максимальное количество одновременно обрабатываемых файлов: 3
Для корректной работы программы в файле RTF не может быть больше одного шрифта. 
В остальных случаях не гарантируется сохранения шрифтов при изменении стиля фрагмента текста.
Поддерживается 2 шрифта default и elephant.

Функция создания *.rtf файлов не требуется поэтому не реализована.

Новое окно    -- Ctrl + Shift + N (Открывает новое окно)
Новый файл    –- Сtrl + N (Открывает панель создания нового файла)
Cохранить     –- Ctrl + S (Сохраняет текущий файл)
Cохранить все –- Ctrl + Shift + S (Сохраняет все файлы)


При закрытии файла предлагается сохранить/закрыть, не сохраняя/отмена.  Что логичнее чем при закрытии окна.
Все содержимое папки с проектом является ее частью не может быть изменено пользователем (в том числе файлы rtf и txt). При изменении, удалении, стороннем открытии любого файла нарушается целостность программы, и корректность работы не гарантируется. В том числе при открытии любого файла .txt или .rtf из папки проекта дальнейшая или действующая работа программы не гарантируется. (Залазишь в системные файлы = твои проблемы)
